🤖 FCE ChatBot
Un bot simple e inteligente para hacer consultar en relacion a materias correlativas, optativas y electivas en la Facultad de Ciencias Económicas (UBA) 🎓🇦🇷
Desarrollado para ayudar a estudiantes de grado de manera clara y accesible.
Proyecto desarrollado en Python con frontend en Streamlit. Disponible en español.

▶️ Tecnologías utilizadas
-  Lógica y datos: Python, Pandas
-  Frontend: Streamlit + CSS personalizado
-  Deploy: Netlify

▶️ ¿Cómo ejecutarlo localmente?
git clone https://github.com/SOFIAMORETTI01/Portafolio.git
cd Portafolio/bot_project
pip install -r requirements.txt
streamlit run bot_correlativas.py

▶️ Version online
https://portfolio-sofia-moretti.netlify.app/

🤖 FCE ChatBot
A simple and smart chatbot to check subject prerequisites, electives, and optional courses at the Faculty of Economic Sciences (UBA) 🎓🇦🇷  
Built to help undergraduate students in a clear and accessible way.  
This project was developed in Python with a Streamlit frontend. Available in Spanish.

▶️ Technologies used  
-  Logic & data: Python, Pandas  
-  Frontend: Streamlit + custom CSS  
-  Deployment: Netlify

▶️ How to run it locally  
git clone https://github.com/SOFIAMORETTI01/Portafolio.git  
cd Portafolio/bot_project  
pip install -r requirements.txt  
streamlit run bot_correlativas.py

https://portfolio-sofia-moretti.netlify.app/
